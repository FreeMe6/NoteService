前提需要安装：

1） NodeJs
2)  Git
--------------------------------------------------------------------
首先安装上述依赖软件，然后进行主目录下

注意，不要解压放置在中文目录下！
--------------------------------------------------------------------
此时，你可以执行 install.sh 脚本来初始化
也可以按照下面的方式来初始化库
--------------------------------------------------------------------
首先，在server目录下右键，点击 [Git Bash Here] 菜单打开命令界面；
执行命令：
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install formidable fs http url path sqlite3 --save
--------------------------------------------------------------------

然后执行run.sh开启后台服务，然后双击index.html打开页面登录即可。
--------------------------------------------------------------------
默认创建账号：
username: root
password: root
